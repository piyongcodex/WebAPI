# Generic Web API (ASP.NET Core + Clean Architecture)
![build](https://img.shields.io/badge/build-passing-brightgreen)
![version](https://img.shields.io/badge/version-v1.0.0-blue)

A simple **Generic Web API** built with **ASP.NET Core**, **C#**, and **Clean Architecture**.

Designed for learning and practicing:

- Clean Architecture
- Repository Pattern
- FluentValidation
- Dependency Injection
- Result Pattern
- Separation of Concerns

---

## How To Use

To clone and run this application, you'll need **Git** and **.NET SDK** installed on your computer.

From your command line:

```bash
# Clone this repository
git clone https://github.com/piyongcodex/WebAPI.git

# Go into the repository
cd your-repo-name

# Run the project
dotnet run --project src/Api
